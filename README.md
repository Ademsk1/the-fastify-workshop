# The Fastify Workshop

![ci](https://github.com/nearform/the-fastify-workshop/workflows/ci/badge.svg)

## Requirements

- Node LTS
- docker
- docker-compose

## Setup

- `npm i`
- `npm run db:up`
- `npm run db:migrate`

## Running the modules

- `cd src/step-{n}`
- check each module's README file to see which scripts are available

## Presenting

- `npm start`
